redis部署到Linux

redis【内存中的数据结构存储系统，它可以用作数据库、缓存和消息中间件】

【1、 安装 gcc】
yum install -y gcc g++ gcc-c++ make

【2、 解压 redis】
tar zxvf redis-3.0.0.tar.gz
cd redis-3.0.0

【3、 执行 make 编译命令】
make MALLOC=libc

安装。make install PREFIX=/usr/local/redisbin 【安装后会在redisbin目录下生成一个bin目录，里面存放的是redis的可执行程序】

【4、 启动关闭 redis】

【4.1 直接在redis安装目录下启动关闭】
启动 src/redis-server & （添加 &执行后台启动 ）
关闭 src/redis-cli shutdown

【4.2 install 后的启动关闭，切换到install的目录】

【启动】bin/redis-server redis.conf 【redis.conf:redis安装目录下的copy文件，改demonize yes即可后台启动】
【关闭】bin/redis-cli shutdown

【5、 查看 redis 是否启动】
ps -ef | grep redis 查看进程
