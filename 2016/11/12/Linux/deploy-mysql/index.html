Linux安装MySQL常见设置

配置系统启动时启动某程序：

加入到系统服务：chkconfig --add 【服务名】mysql
自动启动：chkconfig 【服务名】mysql on
查询列表：chkconfig

开启远程访问:

登录：
mysql -uroot –p[password]

设置远程访问（使用root密码）：
    grant all privileges on *.* to 'root' @'%' identified by 'xiaohe'; 
    flush privileges;

防火墙打开3306端口
    /sbin/iptables -I INPUT -p tcp --dport 3306 -j ACCEPT
    /etc/rc.d/init.d/iptables save
    /etc/init.d/iptables status
